# 👻 GhostMeter - Installation Rapide

## Méthode 1: Copier les fichiers manuellement

### 1. Créer le projet
```bash
mkdir ghostmeter
cd ghostmeter
npm init -y
```

### 2. Installer les dépendances
```bash
npm install next@15 react@19 react-dom@19 framer-motion lucide-react z-ai-web-dev-sdk tailwindcss@4 typescript @types/react @types/react-dom
```

### 3. Créer la structure
```bash
mkdir -p src/app/api/analyze src/components/ui src/hooks public/icons
```

### 4. Créer les fichiers de configuration
- next.config.ts
- tsconfig.json
- tailwind.config.ts
- vercel.json

### 5. Copier le code source
- src/app/page.tsx (page principale)
- src/app/api/analyze/route.ts (API IA)
- src/app/layout.tsx (layout)
- src/app/globals.css (styles)

### 6. Tester et déployer
```bash
npm run dev
# Puis déployer sur Vercel
```
