## 🛠️ Outils de Production Recommandés

### Motion Design Professionnel
| Outil | Usage | Prix |
|-------|-------|------|
| **After Effects** | Animation pro | Abonnement Adobe |
| **Cavalry** | Motion design 2D | ~$40/mois |
| **Figma + Smart Animate** | Prototypes animés | Gratuit |

### IA Vidéo (Alternative)
| Outil | Forces | Limites |
|-------|--------|---------|
| **Runway Gen-2** | Qualité, contrôle | Texte imprécis |
| **Pika Labs** | Rapide, gratuit | Moins de contrôle |
| **Kling AI** | Nouveau, performant | File d'attente |

### Workflow Recommandé
```
1. Générer animations SANS texte (IA vidéo)
2. Exporter en PNG sequence / Vidéo
3. Ajouter texte en post-production (AE/Premiere)
4. Exporter en H.264 + Boucle
```

## 📐 Formats de Sortie

### TikTok / Instagram Reels / YouTube Shorts
```
Format: 9:16 (vertical)
Résolution: 1080x1920px
FPS: 60
Durée: 5-8 secondes
Codec: H.264
Bitrate: 8-15 Mbps
```

### YouTube (si nécessaire)
```
Format: 16:9 (horizontal)
Résolution: 1920x1080px
FPS: 60
Durée: 15-30 secondes
```

## 🎵 Design Sonore Global

### Palette Sonore GhostMeter
- **Tonalité** : Mystérieux mais accessible
- **Instruments** : Synthés doux, plucks, bells
- **Ambiance** : Futuriste, magique, bienveillant

### Bibliothèque Sonore Recommandée

- **Artlist** : Qualité pro, forfait simple

## 📊 Métriques Cibles

### Objectifs Vidéos
| Métrique | Objectif |
|----------|----------|
| **Watch time** | >80% de la durée |
| **Engagement** | >5% likes/comments |
| **Partages** | >1% des vues |
| **CTA clicks** | >0.5% |

### A/B Testing Suggéré
- Version avec/sans voix off
- Différentes couleurs de CTA
- Longueurs variables (5s vs 7s)
