### Description
Écran de smartphone montrant une interface de messagerie. En haut, un avatar de coach IA (fantôme avec casquette).

### UI Initiale
```
┌─────────────────────────────────┐
│  👻 Coach Relationnel           │
│  ████████ En ligne              │
├─────────────────────────────────┤
│                                 │
│                                 │
│                                 │

│                                 │

│                                 │
│                                 │
├─────────────────────────────────┤
│  [💬 Pose ta question...]       │
│                          [_send] │

### Animation
1. L'interface apparaît avec un léger fade-in
2. L'avatar du coach pulse doucement
3. Le placeholder clignote

## 📱 Frame 2 - La Question (0.5-1.5s)

### Description
L'utilisateur pose sa question.

### UI
```
│                                 │
│                    ┌──────────┐ │
│                    │ Il m'a   │ │
│                    │ vue mais │ │

### Animation

2. La bulle apparaît depuis la droite avec un léger rebond
3. Petit "sent" sound

## 📱 Frame 3 - La Réflexion (1.5-2.5s)

### Description
Le coach réfléchit à la réponse.

### Animation de l'Avatar Coach
```
     👻
   ~ 💭 ~
   . . .
```

### Animation
1. L'avatar du coach change d'expression (réflexion)
2. Points de suspension qui orbitent autour de l'avatar
3. Particules violettes qui flottent
4. Effet de "réflexion IA" (petits circuits lumineux)

### Effets sonores
- "Hmm" pensif
- Bruit de "traitement" subtil

## 📱 Frame 7 - L'Animation Cœur (5.5-6s)

### Description
Animation symbolique de guérison.

### Animation du Cœur
```
Frame 1:    💔  (cœur brisé)
Frame 2:    💔  (fissures)
Frame 3:    💗  (réparation)
Frame 4:    💖  (cœur brillant)
```

### Déroulement
1. Un cœur brisé 💔 apparaît au centre
2. Les fissures se referment (animation)
3. Le cœur se reconstitue en 💖
4. Petites étoiles scintillent autour

### Message final
```
        💖

  "Tu vaux mieux que ça."

      👻 GHOSTMETER
```
